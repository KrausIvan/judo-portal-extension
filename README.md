**Zásady ochrany osobních údajů – Judo rozhodčí → Kalendář**

Poslední aktualizace: 23.03.2026

Toto rozšíření pro Chrome je určeno pro rozhodčí Českého svazu judo.

Jaké údaje zpracováváme:
- E-mailová adresa Google účtu se používá výhradně pro přihlášení ke Google Calendar API. Není ukládána ani přenášena.
- Data z portálu rozhodčích: rozšíření čte názvy turnajů, data a role z otevřené stránky portálu. Tyto údaje se přenášejí výhradně do osobního Google Kalendáře uživatele, nebo se stáhnou jako .ics soubor na jeho zařízení.

Jaké údaje nesbíráme: hesla k portálu, osobní údaje nad rámec Google přihlášení, historii prohlížení. Žádná data nejsou sdílena s třetími stranami.

Kde jsou data uložena: veškerá data zůstávají v prohlížeči uživatele a v jeho osobním Google Kalendáři. Rozšíření nepoužívá žádný externí server.
